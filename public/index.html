色覚シミュレーション（Machado 2009 ベース CVD モデル）

・ご利用のブラウザや環境によっては、クリップボードからの貼り付けが使えないことがあります。
・その場合は「ファイルを選択」から画像を読み込んでください。
・グラフや画面キャプチャなどを読み込むと、「色に頼らない情報の伝え方」を考えるヒントになります。
0 に近いほど元の画像に近い色、1 に近いほど各タイプの色の特徴を強くシミュレートします。
実際の見え方は人によってちがうため、「だいたいの傾向を知るための目安」として使ってください。
このツールについて

このページでは、画像の「色の見え方のちがい」をシミュレートできます。
Protan / Deutan / Tritan は、色の感じ方に関するいくつかのタイプをモデル化したものです。 実際の見え方には、個人差があることにご注意ください。

使い方

  1. クリップボードに画像をコピーするか、「ファイルを選択」から画像ファイルを選びます。
  2. 読み込むと、左上の「オリジナル」に元の画像が表示されます。
  3. スライダー（severity）を動かすと、Protan / Deutan / Tritan 各パネルの見え方が変わります。

見え方について

  • このシミュレーションは、研究に基づくモデルを使った「一例」です。
  • 実際のすべての人の見え方と完全に一致するものではありません。
  • 医療的な診断や判定を行うことを目的としたものではありません。

オリジナル

Protan（1型）

Deutan（2型）

Tritan（3型）

色覚シミュレーションは、G. M. Machado らによるモデル “A Physiologically-based Model for Simulation of Color Vision Deficiency” (2009) と、 Python ライブラリ Colour の CVD_MATRICES_MACHADO2010（BSD-3-Clause）に基づいています。
